# Countdown

## Example on white

The following example recreates the datoCMS landing page

```jsx
<LandingProvider>
    <Countdown
        // dark
        // bg='gray.800'
        py='60px'
        heading='We will release soon'
        subheading='Check out in a few days'
        deadline={
            new Date(new Date().getTime() + 1000 * 60 * 60 * 24 * 2 + 60 * 3)
        }
        cta={<MailchimpForm url='' />}
    />
</LandingProvider>
```
