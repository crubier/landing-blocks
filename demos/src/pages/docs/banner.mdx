# Banner

## Example on white

The following example recreates the datoCMS landing page

```jsx
<LandingProvider primary='#FF593D'>
    <Banner
        // dark
        heading='Try Prismic Now!'
        subhead='50% discount just for good fellows like you'
        image={<GradientCurtains />}
        cta='try it out'
    />
</LandingProvider>
```
